# 实战案例库

欢迎贡献你或朋友真实发生过的谈薪案例。每个案例都应该:

1. **不暴露真实姓名 / 公司具体部门** —— 用化名
2. **可学习** —— 写出关键决策点和转折
3. **不夸张** —— 真实数字,即便不漂亮

## 模板

```markdown
## 案例 N: <一句话总结>

### 背景
- 候选人:<行业 / 年限 / 学历>
- 目标公司:<规模 / 行业 / 城市>
- 目标岗位:<职级 / 团队>
- 手上的筹码:<其他 offer / 内部转岗 / 稀缺技能>

### 过程
- 初始 offer:xxx
- 期望薪资:xxx
- HR 关键话术:...

### 使用了哪些策略
- S1 / S2 / S3 / S4
- 用了哪个 / 哪个场景剧本

### 最终结果
- 实际拿到:xxx
- 谈判周期:xxx 天
- 关键转折点:...

### 教训
- ...
```

## 现有案例

### 案例 1:校招白菜价如何谈到 SP *(placeholder)*
- 背景:985 本硕,无大厂实习,拿到某厂白菜 28k
- 策略:S3 替代补偿(争取签字费 + 房补)
- 结果:base 不变,签字费谈到 3 个月 base + 房补 5k/月

### 案例 2:跳槽被压职级 *(placeholder)*
- 背景:6 年经验,被新公司定级 P5,要求 P6
- 策略:质疑定级依据 + 要求薪资反映价值差异
- 结果:定级调整到 P6,base 上浮 8%
